🛡 How to Fix a WebRTC Leak

  • Install WebRTC Leak Shield (Chrome) or (Firefox)
  • Firefox: set media.peerconnection.enabled → false in about:config
  • Use a VPN with built-in WebRTC leak protection
  • uBlock Origin → enable "Prevent WebRTC from leaking local IP addresses"

    ℹ️ How to Prevent DNS Leaks

  • Use a VPN with DNS leak protection + kill switch
  • Set trusted DNS manually: 1.1.1.1 (Cloudflare) or 8.8.8.8 (Google)
  • Configure DNS-over-HTTPS / DNS-over-TLS on your router
  • Test at dnsleaktest.org and ipleak.net
